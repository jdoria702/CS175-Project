# App Name: Try Not To Die!
* Save your character within the time limit to prevent injury or death!
* Upon progression, levels will get harder and harder with shorter time intervals to save your character!
* Earn Coins as you progress to unlock new styles in the shop!

## Specific hardware requirements:
* Android device with Android API 29
* Android emulator can be used from android studio:

https://developer.android.com/studio

## Download and Run Guide
Download APK
* If using physical Android
1. Connect USB cable to device with APK
2. Drag APK build into devices download folder
3. Locate App and play
* Android Emulator
1. Run emulator with API 29
2. Drag the downloaded APK into the emulator
3. Locate App and play

## User Guide: How to Play (Minigames)
* Extinguish Fire Minigame: Repeatedly tap fire to extinguish it and save yourself
* Quick Reaction Button: Properly read the instructions and press the right button before blowing up
* Clean the dirt: Remove the dangerous dirt off your character to prevent illness or death
* Remove the blobs: Tap on all blobs on the screen
* Catch the falling character: Drag the trampoline on the ground to catch your falling character
